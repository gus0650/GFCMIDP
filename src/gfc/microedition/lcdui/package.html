javax.microedition.lcdui replacement.

This package contains classes that are API compatible with those that come with MIDP.

However, in contrast to the original classes, our implementation offers some additional features:

  • customizable look (fonts, colors...)
  • platform independent (runs in an Applet context, on PocketPC etc.)
  • layout will look 100% identical accross all platforms (different Handsets, Applet etc.), because BitmapFont is used instead of system fonts.
